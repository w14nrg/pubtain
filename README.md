# Pubtain one-page website

Complete static website for **Pubtain — Maintain your pub with Pubtain**.

## Upload to GitHub Pages
1. Create a new GitHub repository.
2. Upload every supplied file and folder exactly as arranged.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.

## Before launch
- Search for `hello@pubtain.com` and replace it if needed.
- The form currently opens a prepared email in the visitor's email app.
- The £200 monthly price is editable in `index.html`.
